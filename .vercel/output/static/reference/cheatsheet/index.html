Fungsi text

FungsiDefinisiContohoutput
=LEFTntar ini ngasih hasil data lu dari kiri=LEFT(“Gojo”;2)Go
=RIGHTkek di atas tapi ini nge return dari Kanan=RIGHT(“Gojo”;2)jo
=MIDbukan midlane kek di ML ,ini ngambil teks dari tengah tapi disini ada tambahan 1 variabel buat nambahin berapa teks yang mau di ambil disini gw ambil 2 jadi hasilnya 2 huruf=MID(“Gojo”;2;2)oj
=LENini buat nge return berapa jumlah kata yang ada di data,cocok buat cek NIK yang dimana hurufnya harus tetap,misalnya kurang berarti invalid=LEN(“Gojo”)4
=LOWERkalo rumus ini yang bikin semua text lu jadi huruf kecil semua=LOWER(“GOJo”)gojo
=PROPERini yang paling berguna soalnya ini yang ngebuat huruf jadi Proper alias bener kapitalnya=PROPER(“goJO”)Gojo
=UPPERini biasnya dipake buat lu yang buta atau ga liat text segede gaban tapi ga kebaca karna lu cuma bisa baca yang isinya text kapital semua kyak orang marah-marah=UPPER(“gojo”)GOJO
=REPTini buat ngulangin text sebanyak yang lu mau,disini gw contohin gw mau ngulangin kata 5 kali=REPT(“Gojo”;5)GojoGojoGojoGojoGojo
=CONCATENATEini nge gabungin beberapa text buat di jadiin 1,ini bisa lu kreasiin beda dari =JOIN yang cuma nambah nambah doang,tapi memang ada kekurangannya,ntar lu juga paham dimana ga ada pembatasnya=CONCATENATE(“Blue”;“Red”;“Purple”)BlueRedPurple
=FINDini buat ngasih tau berapa banyak data yang namanya sesuai sama data yang lu kasih,tapi inget ini tuh harus sesuai,soalnya kapital atau salah dikit aja bakal nge return data error atau value ga ada=FIND(A1)”;“Gojo”)1
=SEARCHini mah sama aja kyak yang di atas cuma gw sendiri ga tau apa bedanya intinya begitulah,ntar error nya bisa lu mainin pake rumus logika if atau if else misalnya ga ketemu bakal ngereturn data apa,intinya ini kepake buat double rumus=SEARCH(A1)”;“Gojo”)
=TRIMNAH ini yang paling berguna ,soalnya ini bisa nge hapus spasi yang kelebihan jadi data yang valid,ini bisa lu kombo sama rumus proper jadi jangan lupa sama ni rumus buat ngebersihin data=TRIM(” Gomen Amanai “)Gomen Amanai
=VALUEini buat nge convert text value jadi nomor,tpi kalo lu kerjain di spreadsheet bakal otomatis kok hasilnya jadi value nomor jadi gaperlu lu pake rumus ginian soalnya ga bakal beda hasilnya,sama nanti bakal return error kalo lu ngasih text kyak alfabet=VALUE(“1985”)1985

FungsiDefinisiContohOutput
ANDngasih nilai TRUE misal semua kondisi itu TRUE,misalya ada 1 aja yang false,semuanya false=AND([True];[True];[False])False
ORngasih nilai TRUE kalo ada salah satu nya ada nilai True=OR([True];[False];[False])True
IFini biasanya yang sering dipake buat double rumus,misalnya JIKA kondisi fungsi itu hasilnya true return hasil true nya mau apa,sebaliknya misalnya False return data falsenya mau apa,misalnya lu belajar python atau rust lebih jauh lagi pasti bakal sering ketemu inilu coba aja sendiri ntar .guru/dosen lu juga ntar ngajarin gunanya ifBenar
